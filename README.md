# Nexus

#### 介绍
Nexus是一个简单的rpc中间件，基本实现rpc的主要功能，目前须结合spring使用，其中使用Netty为通信框架，使用kryo作为默认序列化协议，通过spi来选配序列化协议、 负载均衡器、注册中心；
